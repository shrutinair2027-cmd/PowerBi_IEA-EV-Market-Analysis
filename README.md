## EV Market Analysis using Power BI (IEA Data)
# Overview

This project analyzes the global Electric Vehicle (EV) market using data from the International Energy Agency (IEA). The focus is on understanding EV adoption trends, powertrain distribution, and future projections under different policy scenarios.

## Key Features
Data cleaning and transformation using Power Query
Interactive dashboard built in Microsoft Power BI
KPI metrics:
Total EV Sales
Total Stock
Slicers:
Projection Scenario (Historical, APS, STEPS)
Powertrain (BEV, PHEV, FCEV)
Parameter-based analysis for dynamic insights
Analysis Performed
EV adoption trends over time (2010–2025)
Powertrain distribution (BEV, PHEV, FCEV)
Mode-wise analysis (Cars, Buses, Trucks, Vans)
Regional comparison (China, USA, Europe, India)
Scenario-based projections:
APS (Announced Pledges Scenario)
STEPS (Stated Policies Scenario)
## Key Insights
BEVs dominate EV market with highest adoption globally
Passenger cars are the leading vehicle segment
China and USA are the top EV markets
EV growth accelerated rapidly after 2020
APS projections show faster growth compared to STEPS
